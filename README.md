Previsão de Renda e Análise de Gorjetas - Regressão
Este projeto aplica técnicas de regressão linear simples, múltipla e transformações de variáveis para análise de dados.

O objetivo principal é entender os fatores que influenciam a renda e o valor de gorjetas através de modelos estatísticos.

Estrutura do Projeto
O repositório contém três notebooks principais que seguem uma linha evolutiva de aprendizado.

1. Regressão Linear Simples
Foco inicial na relação entre o tempo de emprego e a renda bruta dos indivíduos.

Análise da qualidade do ajuste e comportamento dos resíduos para identificação de padrões não lineares.

2. Transformações em X e Y
Estudo aprofundado utilizando a base de gorjetas para testar modelos logarítmicos e polinomiais.

Comparação de desempenho entre modelos para encontrar o melhor ajuste estatístico para variáveis contínuas.

3. Regressão Múltipla
Construção de um modelo preditivo robusto utilizando múltiplas variáveis explicativas como idade, sexo e posse de imóveis.

Aplicação do método de remoção de variáveis não significantes (p-value) para simplificação e eficiência do modelo.

Tecnologias Utilizadas
O projeto foi desenvolvido em Python utilizando o ambiente Jupyter Notebook.

As bibliotecas principais foram Pandas, Seaborn, Matplotlib, Statsmodels e Patsy.
